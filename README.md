# myway

myway is a static educational guide for Indian students planning their path after Class 10. It groups higher-secondary streams, diploma and vocational routes, entrance exams, careers, and a short aptitude quiz into one browser-based experience.

## What’s inside

- Streams: Science (PCM, PCB, PCMB), Commerce, Arts / Humanities, Diploma, ITI / Vocational, and Paramedical routes.
- Explorer: searchable course cards with filters for stream, duration, and entrance requirement.
- Entrance exams: a browsable list of major Indian entrance tests.
- Careers: entry-level career snapshots with qualification and salary ranges.
- Quiz: a simple 8-question stream suggestion quiz that runs entirely in the browser.
- FAQ and source notes for quick reference.

## Project files

- [index.html](index.html) - page structure and content sections
- [styles.css](styles.css) - visual design and responsive layout
- [app.js](app.js) - rendering, filtering, quiz logic, and modal behavior
- [data.js](data.js) - streams, courses, exams, careers, and FAQ data

## Run locally

Open [index.html](index.html) directly in a browser, or serve the folder with any static file server if you prefer.

## Notes

- The site is static and does not require a build step.
- Content should be verified against official sources before using it for admissions decisions.
